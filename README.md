## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Ubuntu PostgreSQL
```
systemctl status postgresql
```

```
sudo systemctl restart postgresql
```
